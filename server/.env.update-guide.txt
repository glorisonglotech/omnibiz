=====================================================
 OMNIBIZ .env CONFIGURATION GUIDE
 Update your .env file with these values
=====================================================

ESSENTIAL CONFIGURATION (Required for server to work):
-------------------------------------------------------

# MongoDB Connection
MONGO_URI=mongodb://localhost:27017/omnibiz

# JWT Authentication
JWT_SECRET=your_super_secret_jwt_key_change_this_in_production

# Server Configuration
PORT=5000
CLIENT_URL=http://localhost:5173

# Google Gemini AI (REQUIRED for AI features)
GEMINI_API_KEY=your_gemini_api_key_from_makersuite_google_com


EMAIL NOTIFICATIONS (Required for support & notifications):
------------------------------------------------------------

# Email Service (Nodemailer with Gmail)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-gmail-app-password-16-chars
SMTP_FROM=noreply@omnibiz.com

# HOW TO GET GMAIL APP PASSWORD:
# 1. Enable 2FA on your Gmail account
# 2. Go to: https://myaccount.google.com/apppasswords
# 3. Select "Mail" app and "Other" device
# 4. Copy the 16-character password (remove spaces)
# 5. Paste it in SMTP_PASS above


SMS NOTIFICATIONS (Optional - for SMS features):
-------------------------------------------------

# Twilio SMS Service
TWILIO_ACCOUNT_SID=your_twilio_account_sid
TWILIO_AUTH_TOKEN=your_twilio_auth_token
TWILIO_PHONE_NUMBER=+1234567890

# HOW TO GET TWILIO CREDENTIALS:
# 1. Sign up at: https://www.twilio.com/try-twilio
# 2. Get free trial credits
# 3. Copy Account SID and Auth Token from dashboard
# 4. Get a phone number from Twilio


PAYMENT GATEWAYS (Optional - for payments):
--------------------------------------------

# M-Pesa (Safaricom Kenya)
MPESA_CONSUMER_KEY=your_mpesa_consumer_key
MPESA_CONSUMER_SECRET=your_mpesa_consumer_secret
MPESA_BUSINESS_SHORTCODE=174379
MPESA_PASSKEY=your_mpesa_passkey
MPESA_CALLBACK_URL=https://your-domain.com/api/payments/mpesa/callback
MPESA_ENVIRONMENT=sandbox

# PayPal
PAYPAL_CLIENT_ID=your_paypal_client_id
PAYPAL_CLIENT_SECRET=your_paypal_client_secret
PAYPAL_ENVIRONMENT=sandbox


CLOUDINARY (Optional - for file uploads):
------------------------------------------

# Cloudinary Configuration
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret


NGROK (Optional - for local development tunneling):
----------------------------------------------------

# Ngrok Auth Token
NGROK_AUTHTOKEN=your_ngrok_auth_token


=====================================================
 QUICK START (Minimum Required):
=====================================================

For the server to start, you MUST have:
1. MONGO_URI (MongoDB connection)
2. JWT_SECRET (any random string)
3. GEMINI_API_KEY (for AI features)

For notifications to work, add:
4. SMTP_USER (your Gmail)
5. SMTP_PASS (Gmail app password)

Everything else is OPTIONAL!


=====================================================
 HOW TO UPDATE YOUR .env FILE:
=====================================================

1. Open the file:
   nano /home/glorison/projects/omnibiz/server/.env

2. Update the values (don't use quotes):
   WRONG: MONGO_URI="mongodb://localhost:27017/omnibiz"
   RIGHT: MONGO_URI=mongodb://localhost:27017/omnibiz

3. Save and exit:
   Press Ctrl+O, Enter, then Ctrl+X

4. Restart server:
   The server will auto-restart if nodemon is running
   Or press Ctrl+C and run: pnpm run dev
